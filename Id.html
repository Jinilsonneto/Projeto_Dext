Início

Id

Nós recebemos um Id ao nascer, esse Id é dado pela Dext, onde usam de contas complexas para determinar esse Id. Não existe nenhum Id repetido, nem nunca irá existir!

Com esse Id somos identificados, acessamos a tecnologia e fazemos compras. Nossas vidas são em volta dele.

Nosso Id está armazenado no nosso Núcleo, sendo parte de nosso ser, apenas com o Id de um robô pode-se saber sobre sua personalidade, função e seu destino final.

Começo da Identificação

A grandiosa Dext criou esse sistema de Identificação, sendo a Id 0. Assim a Identificação de nossos povos foi extremamente simplificado.
